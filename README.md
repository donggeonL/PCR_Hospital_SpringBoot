## Spring Boot Project

> ### PCR 병원 조회 서비스
> ##### 공공데이터 API를 사용해 PCR검사 , 신속항원검사를 할 수 있는 병원을 찾는 사이트 개발

## 기술 스택
> ##### SpringBoot (IntelliJ) [Server]
> ##### Mustache (Client)
> ##### JavaScript
> ##### 공공데이터 API (건강보험심사평가원_호흡기 진료 지정 의료기관 정보서비스)
> ##### BootStrap
